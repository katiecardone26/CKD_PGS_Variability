# Chronic Kidney Disease Polygenic Score Variability
- This project was an extension of a similar project done on Coronary Artery Disease (https://doi.org/10.1001/jama.2024.23784).
- We assessed the variability of individual-level polygenic score (PGS) risk predictions for PGS that performed similarly at the population-level.
- This analysis was run in PMBB and AOU
